## Notes

- All PWA/favicon icons are derived from the official MSG mark (`icon-msg.png` in Downloads). Do not use Perada or other group logos.
- No third-party stock photography is currently bundled. Hero and section visuals use CSS composition, SVG process diagrams, abstract UI mockups, and brand-safe geometric treatments.
- Do not use logos from other companies abbreviated “MSG” (e.g. msg global solutions).
- When photography is added, document license, source URL, and usage location in this file before production release.
- ProQPay dashboard visuals on the homepage/product page are **product previews**, not production screenshots, unless replaced with approved assets.
